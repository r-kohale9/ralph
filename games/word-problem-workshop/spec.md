# Game Design: Word Problem Workshop

## Identity

- **Game ID:** word-problem-workshop
- **Title:** Word Problem Workshop
- **Class/Grade:** Class 3-5
- **Math Domain:** Number Operations — All four operations
- **Topic:** Creating word problems from math expressions
- **Bloom Level:** L5 Create

## One-Line Concept

Student sees a math expression on screen and must speak or type a real-world word problem that matches it, evaluated by AI for correctness and relevance.

## Target Skills

| Skill                   | Description                                                                |
| ----------------------- | -------------------------------------------------------------------------- |
| Word problem creation   | Construct a meaningful real-world scenario that matches a given expression |
| Operation understanding | Demonstrate deep understanding of what +, -, ×, ÷ mean in context          |
| Multi-step reasoning    | Combine multiple operations into a coherent story (Stages 2-3)             |

## Core Mechanic

- Student sees a math expression (e.g., "5 + 3 = 8").
- Student speaks or types a word problem that matches the expression.
- AI evaluates the response against a rubric for correctness and awards points (3 = correct, 1 = partial, 0 = incorrect).
- A hint and example word problem are available per round if the student needs guidance.

## Rounds & Progression

10 rounds across 3 stages of increasing complexity:

### Stage 1: Single Operations (Rounds 1-4)

- One operation per expression: addition, subtraction, multiplication, division
- Small numbers, straightforward scenarios

### Stage 2: Two-Step Expressions (Rounds 5-7)

- Two operations combined (e.g., "3 × 4 + 2", "24 ÷ 6 + 5")
- Student must weave both operations into one coherent story

### Stage 3: Multi-Step / Compound Expressions (Rounds 8-10)

- Complex expressions with multiple operations and parentheses (e.g., "(4 × 5) + (3 × 2)", "50 - (3 × 8)")
- Requires describing two separate sub-problems within one scenario

| Dimension            | Stage 1                     | Stage 2                 | Stage 3                    |
| -------------------- | --------------------------- | ----------------------- | -------------------------- |
| Operations per round | 1                           | 2                       | 2-3                        |
| Number complexity    | Small (single/double digit) | Medium                  | Larger, with grouping      |
| Cognitive demand     | Translate one operation     | Sequence two operations | Compose compound scenarios |

## Game Parameters

- **Rounds:** 10
- **Timer:** None
- **Lives:** None (open-ended creative task)
- **Star rating:** 3★ = 25+ points, 2★ = 15+ points, 1★ = 1+ points
- **Input:** Voice (speech-to-text) or typed text
- **Scoring:** 3 pts per correct answer, 1 pt per partial, 0 per incorrect (max 30 pts)
