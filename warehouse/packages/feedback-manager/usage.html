🎯 Feedback Manager Demo

A comprehensive feedback system that integrates audio, subtitles, and stickers to provide rich user feedback.

ℹ️ Features:
• Audio playback with Web Audio API
• Dynamic subtitle display
• Animated sticker feedback
• Automatic feedback hiding after min(audio_duration, 10s)
• Graceful error handling
📦 Package Structure:
This is one of three main MathAI packages:
• components - All UI components (subtitle, sticker, timer, popup)
• helpers - Utility helpers (visibility-tracker, etc.)
• feedback-manager - This package (audio + feedback integration)

Usage Code

<!-- Include FeedbackManager -->
<script src="https://storage.googleapis.com/test-dynamic-assets/packages/feedback-manager/index.js"></script>

<script>
// Initialize
await FeedbackManager.init();

// Preload audio files
await FeedbackManager.sound.preload([
  { id: 'audio1', url: 'https://example.com/audio.mp3' },
  { id: 'audio2', url: 'https://example.com/audio2.mp3' }
]);

// Play audio with subtitle only
await FeedbackManager.sound.play('audio1', {
  subtitle: 'Great job!'
});

// Play audio with sticker only
await FeedbackManager.sound.play('audio1', {
  sticker: {
    type: 'LOTTIE',
    json: 'https://assets.lottiefiles.com/path/to/animation.json',
    alignment: 'RIGHT'
  }
});

// Play audio with both subtitle and sticker
await FeedbackManager.sound.play('audio1', {
  subtitle: {
    text: 'Excellent work!',
    title: 'Success'
  },
  sticker: {
    type: 'LOTTIE',
    json: 'https://assets.lottiefiles.com/path/to/animation.json',
    alignment: 'RIGHT'
  }
});

// Play stream with feedback
FeedbackManager.stream.play('stream1', {
  complete: () => console.log('Stream complete'),
  error: (err) => console.error('Stream error:', err)
}, {
  subtitle: 'Streaming audio...',
  sticker: {
    type: 'IMAGE_GIF',
    image: 'https://example.com/loading.gif',
    alignment: 'LEFT'
  }
});

// Feedback auto-hides after min(audio_duration, 10 seconds)
</script>
