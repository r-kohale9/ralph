Visibility Tracker

Track tab visibility and show popup when user becomes inactive.

💡 Try it:
1. Click "Initialize Tracker"
2. Click "Start Tracking"
3. Switch to another tab for 5 seconds
4. Come back to see the popup!

API Helper

Handle communication with the MathAI Core API for game session management.

💡 API Helper Features:
- Submit game session results to backend
- Get student data from backend
- Configurable API endpoints and headers
- Error handling and timeout management

Usage Code

<!-- Single import for all helpers -->
<script src="https://storage.googleapis.com/test-dynamic-assets/packages/helpers/index.js"></script>

<script>
// Wait for helpers to load
window.addEventListener('load', async function() {

  // Access VisibilityTracker (loaded automatically)
  VisibilityTracker.init({
    inactiveTime: 5000, // 5 seconds
    onBecomeInactive: function() {
      console.log('User became inactive');
    },
    onBecomeActive: function() {
      console.log('User became active');
    }
  });

  VisibilityTracker.start();

  // Or use via MathAIHelpers namespace
  MathAIHelpers.VisibilityTracker.start();
});
</script>
