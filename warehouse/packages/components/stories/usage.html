Note: These examples use the actual v2_story JSON format with:
  • data.children[] array structure
  • v2_story.background.{type, value, loop} for backgrounds
  • v2_story.logic.compute_functions[] for branching logic
  • data.button.{text, color, background_color, action} for buttons
  • other.{position, size} for element positioning
  • global_context_variables for shared state

💡 Fullscreen Behavior: When the StoriesComponent is initialized, it automatically takes over the entire viewport (100vw × 100vh) with position: fixed and z-index: 9999. This ensures that only the story is visible on screen at a time, similar to Instagram Stories or Snapchat. When destroy() is called, all fullscreen styles are removed and the container returns to its original state.
